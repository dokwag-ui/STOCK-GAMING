# 주식게임 Dashboard

순수 HTML/CSS/JavaScript로 만든 정적 주식 투자 게임입니다.

## 실행

`index.html`을 브라우저에서 열면 바로 실행됩니다.

## 파일

- `index.html`: 대시보드 화면 구조
- `styles.css`: 화면 스타일
- `data.js`: 실제 주가 기반 테마, 종목, 가격, 이벤트 데이터
- `engine.js`: 게임 계산, 거래, AI 투자자, 순위 로직
- `game.js`: 화면 렌더링, 이벤트 연결, 저장/복구
- `scripts/build-real-data.ps1`: 무료 Yahoo Finance chart endpoint에서 일별 수정종가를 받아 분기별 게임 데이터 생성

## 배포

GitHub 저장소에 파일을 올린 뒤 Vercel에서 정적 프로젝트로 연결하면 됩니다. 별도 빌드 명령은 필요 없습니다.

## 가격 데이터

현재 데이터는 35개 실제 상장사 후보를 포함합니다. 게임 시작 시 각 테마별 5개 후보 중 3개가 랜덤 선택됩니다. 가격은 일별 수정종가를 분기 말 기준으로 샘플링한 뒤 `2012 Q1 = 10,000` 기준으로 정규화했습니다.
